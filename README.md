# sara_simulation
## Dependencies
* ROS Kinetic
* Ubuntu 16.04 LTS
## How to install
* ```cd sara_simulation/catkin_ws```
* ```catkin_make```
## How to launch
* ```roscore```
* ```roslaunch gazebo_ros empty_world.launch```
* ```roslaunch summit_description summit.launch```
* ```rosrun rviz rviz```
